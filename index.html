YIScratchView on Scratch

How do i use it?

Get your project's project ID. The project ID is the numbers on your project page's address bar. Copy it using CTRL+C. After you have done that, go back to this page and click the generate button. A prompt will appear asking for your project ID. Paste your project ID in the prompt(CTRL+V) and click "OK". You did it! You can copy the embed code from the text box and paste it in your website.
